City-of-Bloomington.zabbix-agent
=========
Ansible role for configuring a Zabbix Agent.

Requirements
------------

City-of-Bloomington.linux


Example Playbook
----------------

```yml
- hosts: zabbix-whatever
  become: yes
  roles:
  - City-of-Bloomington.linux
```
